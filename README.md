# 📊 Superstore Sales Analysis Using Python & Matplotlib

## 📌 Project Overview

This project analyzes Superstore sales data using Python and data visualization techniques. The objective is to identify sales trends, profit patterns, customer behavior, shipping performance, and business insights through exploratory data analysis (EDA) and visualizations.

The project was developed using Pandas, NumPy, and Matplotlib in PyCharm.
